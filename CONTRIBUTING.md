# 🙌 Hướng dẫn Đóng Góp vào Dự Án

Chào mừng bạn đến với dự án! Dưới đây là các quy tắc cần tuân thủ để cùng làm việc hiệu quả.

---

## ✅ Yêu cầu chung

- Dùng **Git + GitHub**
- Commit rõ ràng, có cấu trúc (`feat:`, `fix:`, `refactor:`...)
- Tạo Pull Request (PR) cho tất cả các thay đổi

---

## 🧱 Quy tắc branch

- Tạo branch từ `develop`
- Đặt tên theo chuẩn:
  - `feature/<task-name>`
  - `bugfix/<bug-name>`
  - `hotfix/<urgent-bug>`
- Không commit trực tiếp vào `main` hoặc `develop`

---
